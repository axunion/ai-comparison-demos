# Le Ciel Étoilé - 高級レストランWebサイト

架空の高級レストラン「Le Ciel Étoilé（ル・シエル・エトワレ）」のWebサイトです。

## 特徴

- **モダンなデザイン**: 洗練されたUI/UXデザイン
- **レスポンシブ対応**: スマートフォン、タブレット、デスクトップに対応
- **インタラクティブな機能**: スムーズスクロール、メニュータブ切り替え、予約フォーム
- **アニメーション**: スクロール時のフェードイン効果

## セクション

1. **ヒーローセクション**: インパクトのあるメインビジュアル
2. **レストラン紹介**: レストランのコンセプトと特徴
3. **メニュー**: ディナー、ランチ、ワインのメニュー表示
4. **予約フォーム**: オンライン予約機能
5. **お問い合わせ**: 住所、営業時間、連絡先情報
6. **フッター**: ナビゲーションリンクとソーシャルメディア

## 使用方法

1. `index.html`をブラウザで開く
2. 各セクションをスクロールして閲覧
3. メニュータブをクリックしてメニューを切り替え
4. 予約フォームに情報を入力して送信（デモ用）
