# Le Jardin Étoilé - 星降るガーデンレストラン

架空の高級フレンチレストランのWEBサイトです。

## 概要

Le Jardin Étoilé(ル・ジャルダン・エトワレ)は、ミシュラン三つ星を獲得した架空の高級フレンチレストランです。このWEBサイトは、エレガントなデザインと使いやすいインターフェースで、レストランの魅力を伝えます。

## 機能

- **レスポンシブデザイン**: デスクトップ、タブレット、スマートフォンに対応
- **スムーズスクロール**: ナビゲーションメニューからのスムーズな画面遷移
- **アニメーション効果**: スクロールに応じた要素のフェードイン
- **予約フォーム**: バリデーション機能付きの予約システム
- **ギャラリー**: クリックで拡大表示できる画像ギャラリー
- **モバイルメニュー**: スマートフォン向けハンバーガーメニュー

## カスタマイズ

### カラースキーム

`styles.css`の`:root`セクションで以下の変数を編集できます:

```css
--primary-gold: #d4af37;    /* ゴールドアクセント */
--dark-brown: #2c1810;      /* ダークブラウン */
--light-cream: #f8f5f0;     /* ライトクリーム */
--text-dark: #1a1a1a;       /* テキスト（暗） */
--text-light: #666;         /* テキスト（明） */
```

### フォント

Google Fontsを使用しています。変更する場合は`index.html`のlinkタグと`styles.css`のfont-familyを編集してください。

## 実装されている機能

### JavaScript

- スムーズスクロールナビゲーション
- スクロール時のナビゲーションバーの動的変更
- モバイルメニューのトグル
- Intersection Observer APIを使用したスクロールアニメーション
- 予約フォームのバリデーションと送信処理
- ギャラリー画像のモーダル表示
- カウンターアニメーション

### CSS

- CSS Grid & Flexboxによるレイアウト
- CSS変数によるテーマ管理
- レスポンシブデザイン（ブレークポイント: 968px, 640px）
- トランジションとアニメーション
- ホバーエフェクト

## 注意事項

- 画像はプレースホルダー（SVG）を使用しています
- 予約フォームの送信は実際のサーバーに送信されません（コンソールログのみ）
- 実際の運用には、バックエンドシステムとの連携が必要です
